# Humidity Thing

<img src="https://github.com/o-dka/humiditything/blob/main/Pictures/icon.png" />

Climate station using SPI 1306 OLED , Si7021 and Arduino Pro Mini.

##### (All of these devices are 3.3v)
Wiring:
```
Arduino Mini Pro      Sensor           OLED
VCC ================== VCC =========== VCC
GND ================== GND =========== GND
A5  ================== SCL
A4 =================== SDA
D9 =================================== SCK/D0
D10 ================================== SDA/D1
D11 ================================== DC
D12 ================================== CS
D13 ================================== RES
```
