# humiditything
Climate station using Si7021 and Arduino Mini Pro

Wiring:
```
Arduino Mini Pro      Sensor           OLED
VCC ================== VCC =========== VCC
GND ================== GND =========== GND
A4  ================== SCL
A5 =================== SDA
D9 =================================== D1
D10 ================================== D0
D11 ================================== DC
D12 ================================== CS
D13 ================================== RES
```
