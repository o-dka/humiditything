# humiditything
Climate station using SPI 1306 OLED , Si7021 and Arduino Mini Pro

Wiring:
```
Arduino Mini Pro      Sensor           OLED
VCC ================== VCC =========== VCC
GND ================== GND =========== GND
A5  ================== SCL
A4 =================== SDA
D9 =================================== D1
D10 ================================== D0
D11 ================================== DC
D12 ================================== CS
D13 ================================== RES
```
